## 三、文案创作框架

### 核心差异化角度（本公司专属）
1. HR 团队在跑 GitHub
2. 实习生可无限使用自家模型 Token
3. 学到的 workflow 可打包上传 GitHub，天涯海角一键复刻
4. JD 要求会用 AI 工具

### 文案结构模板

```
【开头】反差钩子 — 打破预期（1-2句）
↓
【痛点】普通实习 vs 这里的实习（对比）
↓
【卖点】3个核心福利，每条配一句金句
↓
【职位列表】三大方向，每个方向一句差异化描述
↓
【结尾】行动召唤 + 紧迫感
```

### 传播力设计原则
- 反差感：越反常识越容易被截图转发（如"HR在跑GitHub"）
- 情绪共鸣：打工人都想要「技能随身带走」
- 低门槛互动：评论区扣字引导，降低参与门槛
- 紧迫感结尾：Week XX，限时开放
